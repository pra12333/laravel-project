# laravel-project

create student database form

# technologies

1. laravel
2. mysql
3. vite

## How to setup project

laravel :

```
composer install
php artisan serve

```

vite server:
install node 16 or later to your computer

```
npm install -g vite
npm install
npm run dev
```

## Start using docker

```
docker compose up --build
```
